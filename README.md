# MGR-bush-clearing-interactive-leaflet-map
This map contains soil and vegetation data collected in 2024 from areas in Madikwe Game Reserve that were previously cleared of trees.

#Map reading instructions 
The different layers can be toggled on and off in a box located at the top right corner. When both soil and vegitation data layers are on, it can be slightly difficult to switch between the two, so turn them on and off individaully if you are struggling.
